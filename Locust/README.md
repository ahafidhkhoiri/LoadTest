# Locust 
Locust is an easy to use, scriptable and scalable performance testing tool.

You define the behaviour of your users in regular Python code, instead of using a clunky UI or domain specific language.

This makes Locust infinitely expandable and very developer friendly.

Read more here : https://docs.locust.io/en/stable/what-is-locust.html

# Installation
- Python 3 or Higher
- pip3 install locust

# Runing the testing

locust -f <filename.py>
